## 📝 Kuis Singkat

### 1. Versi Node.js minimum yang dibutuhkan untuk Next.js adalah...

**Jawaban:** _Minimal Node.js versi 18.17 atau lebih baru.
Penjelasan:
Next.js membutuhkan fitur terbaru dari Node.js agar bisa berjalan dengan baik, seperti performa yang lebih cepat dan dukungan sistem modern. Jika menggunakan versi lama, biasanya akan terjadi error saat instalasi atau saat menjalankan project._

### 2. Saat create-next-app, untuk menggunakan JavaScript kita pilih TypeScript = ...

**Jawaban:** _Pilih No.
Penjelasan:
Saat membuat project Next.js, akan ada pertanyaan apakah ingin menggunakan TypeScript. Jika memilih No, maka project akan menggunakan JavaScript biasa. Ini lebih sederhana dan cocok untuk pemula yang belum mempelajari TypeScript._

### 3. File konfigurasi yang muncul saat memilih JavaScript (bukan TypeScript) adalah...

**Jawaban:** _Jawaban:
jsconfig.json
Penjelasan:
File jsconfig.json digunakan untuk mengatur konfigurasi JavaScript, seperti path import agar lebih rapi dan mudah dibaca. Jika menggunakan TypeScript, file ini akan diganti dengan tsconfig.json._
